# Data Structures & Algorithms Repository

A comprehensive collection of Data Structures and Algorithms implementations in C/C++, covering fundamental concepts, sorting algorithms, recursion, and practical applications.

## Repository Structure

### 📁 ADA (Algorithm Design & Analysis)
Core algorithms and recursion implementations:
- **Searching Algorithms**: Binary Search (iterative & recursive)
- **Sorting Algorithms**: Merge Sort, Quick Sort
- **Recursion Series**: 
  - Counting, Factorial, Fibonacci
  - Power of 2, Sum calculations
  - Array sorted checker

### 📁 array-pointer-string
Fundamental C++ concepts:
- Arrays (1D & 2D)
- Pointers and memory management
- Strings
- Functions and classes
- Constructors
- Control structures

### 📁 linked list
Linked list implementations and operations

### 📁 Stack
Stack data structure implementations

### 📁 Queue
Queue data structure implementations

### 📁 Problems
Practice problems and solutions

### 📁 newDSA
Additional data structure implementations

### 📁 webDev

Additional C++ programs and concepts

## Featured Programs

### Sorting Algorithms
- **Merge Sort**: Divide and conquer sorting with O(n log n) complexity
- **Quick Sort**: Efficient in-place sorting using pivot partitioning
- **Binary Search**: Fast searching in sorted arrays

### Recursion Examples
- Factorial calculation
- Fibonacci sequence generation
- Array sum and sorted validation
- Power calculations

### Projects
- **Library Management System** (`library_management.c`): A complete C program for managing library books with checkout/return functionality

## Getting Started

### Prerequisites
- GCC/G++ compiler
- C++11 or higher

### Compilation
```bash
g++ filename.cpp -o filename
./filename
```

For C files:
```bash
gcc filename.c -o filename
./filename
```

## Usage Examples

### Binary Search
```cpp
// Array must be sorted
int result = binarySearch(arr, size, key);
```

### Merge Sort
```cpp
mergeSort(arr, 0, n-1);
```

### Quick Sort
```cpp
quickSort(arr, 0, n-1);
```

## Learning Path

1. Start with **array-pointer-string** for C++ basics
2. Move to **ADA** for algorithm fundamentals
3. Explore **linked list**, **Stack**, and **Queue** for data structures
4. Practice with **Problems** folder
5. Build projects in **webDev**
